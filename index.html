Sublime Text 3 配置LaTex

发表于 2019-03-15

从安装sublime到运行.tex生成pdf

Sublime Text 3 安装：

点这里下载并解压

双击 Sublime Text Build 3176 x64 Setup.exe 安装

安装完成后，在 “C:\Windows\System32\drivers\etc\hosts” 中 添加以下两行：

在Sublime中点击 Help » Enter Lisence ，输入以下licence：

点击 Tools » Install Package Control 安装package control

LaTex配置：

LaTex下载：这里 ，然后双击 install-tl-windows.bat 安装

Sumatra PDF 下载：这里 ，然后安装

打开Sublime，选择 Preferences->Package Control，输入install，选择 install 后输入 LaTex tools选择安装

选择 Tools->Build System 下的 LaTex 选项

然后就可以在Sublime中 Crtl + B 编译 .tex后缀的文件并生成pdf
